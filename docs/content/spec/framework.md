# Framework

*A foundational technology that defines application structure, development patterns, or architectural conventions.*

## Examples

Spring, Django, Ruby on Rails, Angular, Qt, TensorFlow, Apache Spark

## Governance Concerns

Architectural consistency, ecosystem fragmentation, hiring strategy, training requirements, platform standardization

## Includes

* application frameworks such as Spring, Django, Laravel, and ASP.NET Core
* frontend frameworks such as Angular, Vue, and Svelte
* UI frameworks such as Qt and .NET MAUI
* data and processing frameworks such as TensorFlow, PyTorch, and Apache Spark
* integration frameworks such as Apache Camel

## Excludes

* utility libraries with narrow scope
* build systems and deployment tooling
* infrastructure services and managed platforms
* programming, query, and configuration languages

## Disambiguation

### vs Language

Frameworks are built using languages but are not themselves languages.

* Rust is a Language
* Rocket is a Framework

### vs Tool

Tools support engineering workflows. Frameworks shape application architecture.

* Maven is a Tool
* Spring is a Framework

### vs Service

Frameworks run within applications. Services operate as external runtime systems.

* TensorFlow is a Framework
* AWS S3 is a Service

## Classification Rule

If the technology materially shapes application structure, architecture, or development patterns, it is a Framework.
