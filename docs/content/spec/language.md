# Language

*A formal language used to express code, queries, configuration, templates, or structured data.*

## Examples

Python, Rust, Java, SQL, YAML, XML, Terraform HCL

## Governance Concerns

Standardization, interoperability, maintainability, hiring, long-term skill availability

## Includes

* programming languages such as Python, Go, Rust, Java, and COBOL
* query languages such as SQL, SPARQL, and GraphQL
* markup and templating languages such as HTML, XML, Markdown, and Jinja
* configuration and serialization languages such as YAML, TOML, JSON, and HCL

## Excludes

* frameworks and application platforms
* build systems and developer tooling
* infrastructure platforms and external runtime systems
* compilers, transpilers, generators, and linters

## Disambiguation

### vs Framework

A Language defines syntax and semantics. A Framework defines application structure and development patterns.

* Python is a Language
* Django is a Framework

### vs Tool

Tools process, validate, compile, package, or analyze languages.

* SQL is a Language
* GCC is a Tool

### vs Service

Services are external systems used during runtime operation.

* Java is a Language
* PostgreSQL is a Service

## Classification Rule

If the technology primarily expresses logic, structure, configuration, or queries through formal syntax, it is a
Language.
