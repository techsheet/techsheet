**An open-source content management system written in [PHP](../language/php.md), designed for building and
managing websites ranging from simple blogs to complex enterprise portals.**

Joomla is a modular CMS built on a Model-View-Controller architecture with a component-based extension system that
separates core functionality from add-ons. The platform provides a hierarchical access control list, a flexible content
categorization system, and multilingual support as a core feature rather than an add-on. Since version 4, the framework
has been substantially rebuilt on [Symfony](symfony.md) components and modern PHP practices,
adopting [Composer](../tool/composer.md)-based dependency management, namespaced classes, and a dependency
injection container. The templating layer uses a customizable template override system that allows developers to modify
output without altering core files.

Originally forked from the Mambo CMS in 2005, the platform has grown into one of the most widely deployed content
management systems, powering government websites, educational institutions, corporate intranets, and community portals
worldwide. Its extension directory offers thousands of components, modules, and plugins covering e-commerce, social
networking, forums, and document management. Joomla 4 and 5 modernized the platform with a Bootstrap 5-based admin
interface, a Web Services API for headless content delivery, and improved media management, while maintaining backward
compatibility paths for existing extensions.
