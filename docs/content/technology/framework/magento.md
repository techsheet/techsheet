**An open-source e-commerce platform built on [PHP](../language/php.md), designed for large-scale and highly
customizable online stores.**

Magento is a feature-rich e-commerce framework that provides a modular architecture for building and managing online
storefronts. Originally released in 2008 by Varien, the platform was acquired by Adobe in 2018 and rebranded as Adobe
Commerce for its commercial edition. Magento 2, the current major version, is built
on [Symfony](symfony.md) components and uses
[Composer](../tool/composer.md) for dependency management. Its architecture follows a service-oriented design with
a layered module
system, an entity-attribute-value (EAV) data model for flexible product catalogs, and a dependency injection framework
that enables extensive customization without modifying core code.

The platform is widely adopted by mid-to-large retailers and B2B merchants who require complex catalog structures,
multi-store configurations, and advanced pricing rules. Magento's extension marketplace (formerly Magento Connect)
provides thousands of plugins covering payment gateways, shipping integrations, and marketing tools. The open-source
edition (Magento Open Source, formerly Community Edition) remains freely available, while Adobe Commerce adds cloud
hosting, AI-powered merchandising, and B2B-specific features. The platform supports GraphQL and REST APIs for headless
commerce architectures, and its PWA Studio enables progressive web app storefronts.
