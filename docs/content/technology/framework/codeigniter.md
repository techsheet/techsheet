**A lightweight, full-stack [PHP](../language/php.md) framework built for developers who need a simple,
high-performance toolkit for web applications.**

CodeIgniter is an open-source MVC (Model-View-Controller) web framework for [PHP](../language/php.md), known for
its small footprint, minimal
configuration requirements, and straightforward documentation. Unlike heavier frameworks that rely on extensive code
generation or rigid project structures, CodeIgniter emphasizes a nearly zero-configuration approach with loose coupling
between its components. The framework provides a set of core libraries for common tasks such as database abstraction,
form validation, session management, and email handling, while allowing developers to selectively load only the
components they need.

CodeIgniter is widely adopted for small to medium-scale web applications and REST APIs, particularly in environments
where shared hosting constraints or limited server resources make larger frameworks impractical. Originally created by
EllisLab in 2006, the project is now maintained by the British Columbia Institute of Technology. CodeIgniter 4,
a ground-up rewrite released in 2020, modernized the framework significantly by requiring PHP 7.4+, adopting PSR-4
autoloading and namespaces, introducing a redesigned ORM-like Model layer, and adding first-class CLI tooling. It
remains a popular choice in regions and teams that favor explicit, procedural-leaning code over the annotation-driven
or convention-heavy patterns found in [Laravel](laravel.md)
or [Symfony](symfony.md).
