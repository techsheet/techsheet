**A professional integrated development environment built for [Java](../language/java.md) and JVM languages,
developed by JetBrains.**

IntelliJ IDEA provides deep static analysis, context-aware code completion, and automated refactoring across
[Java](../language/java.md), [Kotlin](../language/kotlin.md), [Groovy](../language/groovy.md),
[Scala](../language/scala.md), and other JVM languages. Its indexing engine builds a semantic model of the entire
project, enabling features such as cross-language navigation, data flow analysis, and safe multi-file refactorings that
go beyond simple text transformations. The IDE ships in two editions: a free Community Edition covering core JVM
development, and a paid Ultimate Edition that adds support for web frameworks, database tooling, application servers,
and
enterprise technologies.

IntelliJ IDEA is the foundation of JetBrains' IDE platform, which also powers specialized editors such as Android
Studio (Google's official Android IDE), PyCharm, WebStorm, and Rider. Its built-in support for
[Gradle](gradle.md), [Maven](maven.md), and [Git](git.md) makes it a standard
choice for enterprise JVM projects. JetBrains also develops [Kotlin](../language/kotlin.md), and IntelliJ serves
as
the reference implementation environment for the language. Recent versions have introduced AI-assisted coding features,
remote development via SSH and cloud-based dev environments, and a new UI built on a Kotlin-based rendering framework.
