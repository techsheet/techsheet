**An AI-powered code completion and assistance tool that provides autocomplete, search, and chat capabilities across
major IDEs.**

Codeium, developed by Exacode, uses proprietary large language models trained on permissively licensed code to deliver
inline code suggestions, natural-language search, and an in-editor chat interface. It distinguishes itself from
competitors by offering a free individual tier with unlimited autocomplete, supporting over 70 programming languages,
and providing extensions for more than 40 editors including [Visual Studio Code](vscode.md), JetBrains
IDEs, Vim, and Emacs. The underlying models are optimized for low-latency, context-aware completions that incorporate
repository-level understanding through its Context Awareness system.

Codeium is used by individual developers and enterprise teams seeking AI-assisted coding without per-seat licensing
costs at the basic tier. In 2024, the company launched [Windsurf](windsurf.md), a standalone AI-native
IDE built on a fork of VS Code, and subsequently shifted its brand focus toward Windsurf as the primary product. The
Codeium extension continues to function as the autocomplete engine within the broader Windsurf ecosystem, while
enterprise deployments support self-hosted and on-premises configurations for organizations with strict data residency
requirements.
