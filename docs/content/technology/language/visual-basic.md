**An event-driven, object-oriented programming language from Microsoft, designed for rapid application development on
Windows.**

Visual Basic evolved from the original BASIC language and was first released in 1991 as a graphical development
environment for building Windows applications. The language emphasizes readability and ease of use, with English-like
syntax, optional static typing, and COM-based component integration. Its successor, VB.NET, runs on the .NET Common
Language Runtime and is fully interoperable with [C#](csharp.md)
and [F#](fsharp.md), sharing the same standard library and runtime infrastructure. VB.NET supports
classes, interfaces, generics, LINQ, and async/await, though it retains backward-compatible syntax elements from classic
VB6.

Visual Basic for Applications (VBA) remains embedded in Microsoft Office products and is one of the most widely deployed
scripting languages in enterprise environments, particularly for Excel automation and Access database applications.
Classic VB6, despite being discontinued in 1998, still runs in legacy line-of-business applications across many
organizations. VB.NET is supported through [ASP.NET Core](../framework/asp-net-core.md) for web development,
though Microsoft announced in 2020 that VB.NET would no longer receive new language features,
positioning [C#](csharp.md) as the primary .NET language going forward.
