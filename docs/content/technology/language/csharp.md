**A modern, multi-paradigm programming language developed by Microsoft for building applications on the .NET platform.**

C# is a statically typed, garbage-collected language that supports object-oriented, functional, and generic programming
within a unified type system where all types, including primitives, derive from a common `object` root. It compiles to
Common Intermediate Language (CIL) and runs on the .NET runtime (CoreCLR), providing cross-platform execution on
Windows, Linux, and macOS. The language features properties, events, delegates, LINQ (Language Integrated Query) for
declarative data manipulation, async/await for asynchronous programming, pattern matching, nullable reference types for
compile-time null safety, and value types (structs) for performance-sensitive scenarios. Designed by Anders Hejlsberg at
Microsoft and first released in 2000, C# has maintained a rapid evolution cadence with annual releases standardized
through ECMA-334 and ISO/IEC 23270.

C# is the primary language for .NET enterprise development, powering backend services with ASP.NET Core, desktop
applications with WPF and WinUI, mobile apps with .NET MAUI, and game development through the Unity engine. The
transition from .NET Framework to the open-source, cross-platform .NET (5+) broadened its deployment beyond Windows.
Recent versions (C# 11–13) have introduced raw string literals, required members, collection expressions, primary
constructors for classes, and `ref struct` interfaces, continuing to add expressiveness while maintaining backward
compatibility across a large installed base.
