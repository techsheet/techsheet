**A procedural, statically typed programming language designed for teaching structured programming and reliable software
development.**

Pascal was created by Niklaus Wirth in 1970 as a language to encourage good programming practices through structured
programming and data structuring. It features strong static typing, explicit variable declarations, and a clear
separation between procedures and functions. The language enforces a strict block structure with well-defined scoping
rules and supports enumerated types, subranges, records, variant records, and pointer types. Pascal's design prioritizes
readability and correctness over conciseness, making it one of the most influential languages in computer science
education.

Borland's Turbo Pascal (1983) and later Delphi transformed the language into a commercially significant platform for
application development, particularly on Windows. Object Pascal, the object-oriented extension used in Delphi and Free
Pascal, added classes, inheritance, interfaces, and generics. The Free Pascal compiler and Lazarus IDE maintain an
active open-source ecosystem with cross-platform support. [C#](csharp.md) inherited several design
principles from Pascal through Anders Hejlsberg, who designed both Turbo Pascal/Delphi and C#.
