**The ubiquitous dynamic programming language of the web, powering interactive behavior in browsers and server-side
applications alike.**

JavaScript is a dynamically typed, prototype-based, multi-paradigm programming language originally designed to add
interactivity to web pages. It supports first-class functions, closures, and event-driven programming, with a
single-threaded execution model built around an asynchronous event loop. The language is standardized as ECMAScript,
with annual specification releases introducing features such as modules, async/await, optional chaining, and private
class fields. [TypeScript](typescript.md), a statically typed superset, has become the predominant way
to write large-scale JavaScript applications.

The language runs natively in every major web browser and, through [Node.js](../tool/nodejs.md), on servers and
command-line tooling. Its package ecosystem, centered on [NPM](../tool/npm.md), is the largest of any programming
language. Frontend development is dominated by frameworks such as [React](../framework/react.md),
[Vue.js](../framework/vue.md), [Angular](../framework/angular.md),
and [Svelte](../framework/svelte.md),
while [Express](../framework/express.md) and [Next.js](../framework/nextjs.md) are widely adopted for
server-side rendering and API development. Modern JavaScript engines (V8, SpiderMonkey, JavaScriptCore) employ
just-in-time compilation and advanced optimization techniques, making the language competitive for compute-intensive
workloads beyond its original browser scripting roots.
