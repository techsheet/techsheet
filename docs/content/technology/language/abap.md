**A proprietary, high-level programming language created by SAP for developing business applications on its enterprise
platform.**

ABAP (Advanced Business Application Programming) is a statically typed, procedural language with object-oriented
extensions, designed specifically for large-scale enterprise resource planning (ERP) systems. It features integrated
database access through Open [SQL](sql.md), internal tables as a first-class data structure, and
built-in support for
multilingual text and currency handling. Originally influenced by [COBOL](cobol.md), ABAP evolved from
a report-generation
language into a full application development platform with its own runtime environment on the SAP NetWeaver Application
Server.

ABAP remains the primary language for customizing and extending SAP S/4HANA and earlier SAP ERP systems, with an
installed base spanning thousands of enterprise customers globally. SAP introduced ABAP RESTful Application Programming
(RAP) as its modern development model, emphasizing a clean separation of concerns with entity manipulation language
(EML) and metadata-driven UI generation through Fiori Elements. ABAP Cloud, the cloud-optimized variant, restricts
access to a curated set of released APIs, enforcing extensibility guidelines for SAP's Business Technology Platform.
