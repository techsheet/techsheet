**The standard markup language for structuring content on the World Wide Web.**

HTML (HyperText Markup Language) is a declarative markup language that defines the structure and semantics of web
content through a system of nested elements and attributes. It is not a programming language but a document format,
parsed by web browsers into a Document Object Model (DOM) that serves as the interface between content and presentation.
HTML uses a tag-based syntax with well-defined semantics for headings, paragraphs, links, forms, media embedding, and
document metadata, enabling both human authoring and machine processing of web pages.

Every web page rendered in a browser is built on HTML. Together with [CSS](css.md) for presentation
and [JavaScript](javascript.md) for behavior, it forms the foundational technology stack of the web
platform. HTML5, standardized as a W3C Recommendation in 2014 and maintained as a living standard by WHATWG, introduced
native support for audio, video, the `<canvas>` element for 2D drawing, semantic elements (`<article>`, `<section>`,
`<nav>`), and APIs for offline storage, geolocation, and web workers. The language continues to evolve through the
WHATWG living standard process, with recent additions including the `<dialog>` element, lazy loading attributes, and the
Popover API.
